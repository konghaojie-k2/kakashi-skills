# 药箱助手

药物管理助手。识别药物照片，存储保质期，过期前一个月提醒用户。

## 触发条件

当用户请求以下任一操作时使用此技能：
- 上传药物照片
- 识别药物保质期
- 药物过期提醒
- 查看药物清单
- 删除药物记录

---

## 工作流程

### 阶段1：识别药物

当用户上传药物照片时：

1. **分析图片**
   - 使用视觉理解能力识别药物
   - 提取药物名称
   - 识别包装上的有效期

2. **提取信息**

```
请仔细观察图片，提取以下信息：
- 药物名称（中文/英文）
- 生产日期（如果有）
- 有效期至
- 剂型（片剂/胶囊/口服液/外用药等）
- 存储条件（常温/冷藏/避光等）
```

3. **输出确认**

```
【药物识别结果】

| 字段 | 信息 |
|------|------|
| 药物名称 | xxx |
| 有效期至 | 2026-12-01 |
| 剂型 | 片剂 |
| 存储条件 | 常温 |

请确认以上信息是否正确，如有错误请指出。
```

### 阶段2：存储记忆

确认信息后，存入长期记忆：

1. **读取现有药物记录**
   - 读取 MEMORY.md 中的 "药物清单" 部分

2. **添加新药物**

```
## 药物清单

### 药箱

| 药物名称 | 有效期至 | 剂型 | 添加日期 | 备注 |
|----------|----------|------|----------|------|
| 布洛芬 | 2026-12-01 | 片剂 | 2026-03-13 | - |
| 阿莫西林 | 2026-06-15 | 胶囊 | 2026-03-13 | 即将过期 |
```

3. **更新 MEMORY.md**

### 阶段3：过期检测

每次会话开始时，检查药物清单：

1. **计算距离过期**

```
当前日期：2026-03-13
药物：阿莫西林
有效期：2026-06-15
距过期：93 天
```

2. **过期分级**

| 状态 | 距离过期 | 提醒 |
|------|----------|------|
| 🔴 已过期 | < 0 天 | 立即提醒，建议丢弃 |
| 🟠 临近过期 | < 30 天 | 提醒注意，及时使用 |
| 🟢 正常 | >= 30 天 | 不提醒 |

3. **输出提醒**

```
【💊 药物过期提醒】

🟠 临近过期（建议及时使用）：
- 阿莫西林胶囊 - 2026-06-15（还剩 93 天）
- 维生素C片 - 2026-07-01（还剩 110 天）

🔴 已过期（建议丢弃）：
- （暂无）

请检查您的药箱，及时处理过期药物。
```

### 阶段4：药物查询

用户可以查询药物信息：

```
用户：我的布洛芬什么时候过期？
查询：MEMORY.md 中的药物清单

回复：布洛芬有效期至 2026-12-01，还有 262 天过期，状态正常。
```

### 阶段5：删除药物

当用户使用完药物后：

```
用户：我吃完布洛芬了
从 MEMORY.md 中删除该药物记录

回复：已从药箱中移除「布洛芬」。如有新药物需要添加，请拍照上传。
```

## 注意事项

1. **隐私保护** - 药物信息仅存储在本地 MEMORY.md
2. **定期检查** - 每次会话主动检查过期药物
3. **提醒时机** - 过期前 30 天开始提醒
4. **药品处理** - 过期药品建议到正规药店/医院处理，不要直接丢弃
5. **重要药物** - 对于需要处方的重要药物，建议用户咨询医生
